# Configuration

DisMusicPresence will use a local settings file named `dmp.settings`.

The settings file is intended for local machine configuration, tokens, source preferences, formatting templates, and other runtime options once implementation begins.

## Planned Configuration Areas

- Discord presence behavior.
- Enabled media sources.
- Apple Music formatting.
- Plex/Tautulli connection settings.
- Presence text templates for listening and watching states.

Do not commit `dmp.settings` to git.
